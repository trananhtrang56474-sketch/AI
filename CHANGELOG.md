# 更新日志 (Changelog)

## 2026-01-02 (架构重构与交互优化)

### 🎨 前端界面重构
- **布局优化**：移除了 `ChatPage` 和 `App` 中重复的侧边栏，统一使用全局 `Sidebar` 控制。
- **侧边栏升级**：
  - 整合了“历史记录列表”功能，删除了冗余的 `HistoryList.vue`。
  - 实现了**折叠/展开**功能（汉堡菜单），收起时自动变窄。
  - 修复了父容器宽度写死的问题，实现侧边栏与右侧内容的**响应式联动**。
- **纯净聊天页**：`ChatPage` 现在只负责显示内容，不再包含导航逻辑。

### ⚡️ 逻辑功能增强
- **路由控制**：实现了通过 URL 参数 (`?session_id=xxx`) 驱动聊天窗口加载历史记录。
- **组件通信**：引入 `eventBus.js`，解决了“右侧新建对话 -> 左侧列表自动刷新”的跨组件通信问题。
- **交互细节**：优化了新建对话的流程，点击“开始新对话”会自动清空当前窗口并重置路由。

### 🗑️ 代码清理
- 删除了废弃的 `HistoryList.vue`。
- 清理了 `App.vue` 和 `ChatPage.vue` 中的冗余代码。
